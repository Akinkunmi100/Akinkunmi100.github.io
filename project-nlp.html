Automated Document Classifier

Engineered a BERT-based classification pipeline that reduced manual document processing time by 40%, saving the operations team 20+ hours per week.

The Challenge

The client was receiving 5,000+ support tickets and legal documents daily. Manual sorting resulted in a 48-hour response delay and high operational costs. They needed an automated solution to route tickets to the correct department instantly with high accuracy.

The Solution

I developed an end-to-end NLP pipeline using Python and TensorFlow. Moving beyond simple keyword matching, I fine-tuned a pre-trained BERT model to understand the context and sentiment of the documents.

  • Implemented text preprocessing with NLTK (tokenization, lemmatization).
  • Trained the model on a dataset of 50k historical documents.
  • Deployed the model as a REST API using FastAPI and Docker containers.

The Results

40%Time Saved
95%F1-Score
<200msLatency
