Revenue Intelligence Dashboard

Designed an interactive Power BI dashboard that identified $120k in underperforming inventory, enabling executive leadership to pivot Q4 strategy.

The Challenge

The retail client was relying on static Excel spreadsheets for weekly reporting. This latency meant they were missing sales trends and overstocking low-demand items, tying up capital in stagnant inventory.

The Solution

I connected their SQL database directly to Power BI to create a live-updating dashboard. I utilized complex DAX measures to calculate Month-over-Month growth and inventory turnover ratios.

  • Modeled data using Star Schema for optimized query performance.
  • Created dynamic drill-downs allowing managers to view sales by region, category, and SKU.
  • Implemented Row-Level Security (RLS) to ensure regional managers only saw their data.
