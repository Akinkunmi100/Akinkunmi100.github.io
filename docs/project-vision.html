Computer Vision

Manufacturing Quality Control System

Deployed a YOLOv8 object detection model to edge devices, decreasing production line defect rates by 15% in real-time testing.

The Challenge

A manufacturing plant relied on human visual inspection for quality control. Fatigue led to a high error rate (3%), allowing defective parts to reach assembly, causing costly recalls.

The Solution

I implemented an automated visual inspection system using Computer Vision. By training a YOLOv8 model on 2,000 labeled images of defective vs. healthy parts, I created a robust detection system.

  • Used OpenCV for image augmentation (rotation, lighting) to improve model robustness.
  • Optimized the model using TensorRT for low-latency inference on edge devices (Jetson Nano).
  • Integrated with the factory's existing MQTT alert system.
