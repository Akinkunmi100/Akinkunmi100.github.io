Automated Document Classifier

How I reduced manual document processing time by 40% using BERT and Python, saving the operations team 20+ hours per week.

The Challenge

The client was manually sorting 5,000+ support tickets daily. This resulted in a 48-hour response delay and high operational costs. They needed an automated solution to route tickets to the correct department instantly.

The Solution

I engineered a text classification pipeline using Python and TensorFlow. By fine-tuning a BERT model, I achieved contextual understanding of the tickets, far surpassing traditional keyword matching.

  • Data cleaning with Pandas & NLTK.
  • Model training on AWS Sagemaker.
  • API deployment using Docker & FastAPI.
